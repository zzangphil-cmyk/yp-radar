# 연구노트 — SBV 방법론의 주식 레이더 적용 타당성

- **작성**: 2026-06-28 · **성격**: 피저빌리티(타당성) 검토. 실서비스 반영 아님.
- **연구 질문**: 시청률 예측용 SBV(멀티모달 장면벡터 + 학습 brain) 방법론을 주식 종목 레이더에 적용하면 무엇이 transfer되고 무엇이 함정인가?

---

## 1. 배경 — SBV란

한국 방송 콘텐츠의 장면별 **1,340D 멀티모달 벡터**(Vision 523 / Audio 5 / Language(SBERT) 779 / Temporal 5 / Program 16 / Context 12)로 **시청률(Reaction)**을 예측하는 학습 brain. 검증된 자산: New Brain R²0.985, phil brain R²0.95, multimodal fusion gain +0.1157, 광고 도메인 transfer 96%, 50~60% engagement 격차 14.35x.

**가장 중요한 정직한 통찰**: phil brain 결정의 **85%는 편성·맥락(Context 44%+Program 35%+ch 5%), 연출(Vision)은 10%**. 즉 *겉보기 연출 직관은 within-channel 통제에서 사라진다*(몽타주·midpoint·H-D1 등 다수 직관 RN-009/010에서 기각). → "편성 예측기지 연출 평가기 아님."

## 2. 방법론 Transfer 분석

| SBV 자산 | 주식 transfer? | 근거 |
|---|---|---|
| 시청률 예측 R²0.98 | ❌ **불가** | 시청률은 콘텐츠→반응 준결정론(고SNR). 주식 일일수익은 노이즈 지배(상한 R²~0.01–0.05), 효율시장·비정상성·재귀성 |
| 상대화(채널별 z) | ✅ | 종목별 평소분포 대비 RVOL·z (현 레이더와 정합) |
| **그룹 기여도 분해** | ✅ **핵심** | 'Program+Context 68.5% 지배' → 주식: 시장·섹터 vs 종목고유 분해 |
| Multimodal fusion | ✅ (측정 한정) | fusion gain은 *측정값*(+0.1157). 주식도 가정 금지·측정만 |
| fail-fast / NULL 폐기 | ✅ **가장 값짐** | 그럴듯한 직관이 통제 후 사라지는지 검증하는 규율 |
| Living dataset √N | ❌ | 주식 비정상성 → 데이터가 쌓일수록 낡음(√N 음전 가능) |
| unique reward asset | ⚠️ 조건부 | 가격엔 없음. 토스 리테일 *행동* 데이터라면 해자 후보 |

### 주식판 '편성 ≠ 연출' 가설 (라운드테이블 핵심)
- **'연출'(겉보기·함정)** = 차트패턴·캔들모양·이격도 → 섹터·매크로 통제하면 잔차로 소멸 위험 1순위.
- **'편성·맥락'(지배)** = 섹터 로테이션·지수·금리·수급·시총밴드.
- 예측: *레이더가 띄우는 이상치 대부분은 "테마 전체가 같이 뜬 것"이지 종목 고유 신호가 아니다.* → §3에서 실증.

## 3. 실증 (A) — "왜 떴나" 기여도 분해

등락률(종가 기준)을 `시장 + 섹터(테마) + 종목고유` 3성분으로 가법 분해. 데이터: 토스 일봉 30거래일 × 50종목. (스크립트 [prototype/decompose.mjs](prototype/decompose.mjs), 결과 [prototype/result.md](prototype/result.md))

**결과**
- 이상치(anomaly≥0.45) **170건** 중 **시장·섹터 주도 ≈ 50% / 종목 고유 ≈ 50%**. 평균 시스템(시장+섹터) 점유율 51%.
- 예시(최신일): SK −9.83%는 **시장·섹터 주도**(고유 −4.35 ≈ 시장), 피에스케이 +13.87%·SK스퀘어 −10.88%는 **종목 고유**(고유 잔차 큼).

**해석**: 가설 **지지**. 현 레이더는 "테마가 같이 뜬 것"을 이상치로 절반쯤 오탐 중. **시장·섹터 통제 후 고유 이탈**을 구분하면 정직성·유용성이 크게 오른다 — 예측이 아니라 *설명/탐지*의 개선. 추가 데이터 0으로 즉시 가능.

## 4. 설계 (B) 요약 — 상세 [03-B-설계.md](03-B-설계.md)

- **아키텍처**: 레이더 Y축을 raw 등락률 → **시장·섹터 통제 후 고유 잔차(σ 정규화)**로 교체. 블립 6→9요소 append(`retSpecific, ledBy, shareSpecific`). 색=`ledBy`(고유=풀채도/시장동반=회색), 크기=`anomaly×shareSpecific`. 바뀌는 파일 4개.
- **Stock Behavior Vector 8그룹**: G1 가격액션 · G2 거래량미시구조 · G3 변동성 · G4 모멘텀 · G5 섹터/시장맥락 · G6 캘린더/이벤트 · G7 수급(외인/기관) · G8 뉴스(SBERT). G1~G5(+G6부분)는 현 데이터 가능, **G7·G8은 데이터 추가 필요**. 투자 순서 G5 > G2 > G7 > G8.
- **분해 산식(권고)**: 매일 횡단면 OLS `ret = α + β·market + Σγ·sector + ε`, 잔차 ε = 종목 고유. (베타조정보다 비정상성·lag에 강건)
- **fusion 측정 프로토콜**: 대리 타깃(이후 실현변동성 급등 / 절대수익 상위5%) 대비 채널 추가 시 ΔAUROC로 lift 측정. 채널당 +0.03 미만이면 폐기.

## 5. fail-fast 게이트 (G1 — A→M 통과 조건)
1. **Look-ahead 차단** — 모든 baseline/β는 t 이전 데이터만(당일종가로 당일신호 금지).
2. **Walk-forward** — rolling 재추정, in-sample 전체적합 금지.
3. **시장통제 후 잔존** — A의 "고유 50%"가 윈도우 무관하게 재현.
4. **Survivorship** — 상폐 포함 유니버스(불가 시 '생존편향 있음' 명시).
5. **거래비용/유동성** — lift 평가 시 스프레드·슬리피지 차감, 소형주 필터.
6. **NULL 폐기 기준** — walk-forward AUROC<0.55 전체폐기, 채널 ΔAUROC<0.03 그룹제외, Precision@20<30% 실용불가.

## 6. 결론 · 권고

- **이식되는 것은 예측력이 아니라 *방법론*** — 상대화 · 기여도 분해 · fusion 측정 · fail-fast. 레이더는 "주가 예측기"가 아니라 **"왜 떴나(시장·섹터 vs 고유)를 분해하는 정직한 이상 스캐너"**가 된다.
- **A**: 즉시 가능·고가치. (다음 단계: 구현 시 위 데이터계약·UI 반영)
- **M(fusion)**: 분봉·수급·뉴스 적재가 *하드 전제*. fusion은 측정용. G1 통과 후.
- **P(예측)**: 봉인. 별도 재합의 없이는 진입 금지. R²·매매신호 framing 금지. **이상 ≠ 매매신호 / 투자자문 아님.**
