# B — 심층 설계 (아키텍처 로드맵 + Stock Behavior Vector 신호/피처)

> radar-architect(아키텍처) + signal-quant(신호/정량) 산출. 피저빌리티 설계 — 구현 전 게이트 통과 전제.

---

## Part 1. 아키텍처 / 로드맵 (radar-architect)

### 1-1. A(기여도 분해) 통합 구조
- **핵심 한 줄**: 레이더 Y축을 raw 등락률 → **시장·섹터 통제 후 고유 잔차(σ 정규화)**로 바꾸고, 블립을 6→9요소 append-only 확장.
- **블립 계약**: `[i, x, y, anomaly, relVol, retPct, retSpecific(6), ledBy(7), shareSpecific(8)]`
  - `ledBy`: 0=고유 / 1=섹터동반 / 2=시장동반 (지배 성분).
- **radar-frames.json 메타**: `axes.y = "시장·섹터 통제 후 고유 수익률(σ)"`, `decomposition` 블록 추가.
- **UI**: 색=`ledBy`(고유 풀채도/시장동반 회색), 크기·정렬=`anomaly×shareSpecific`, 패널에 시장/섹터/고유 분해 노출.
- **바뀌는 파일**: `build-radar.mjs`·`radarData.ts`(signal-quant) / `StockRadar.tsx`(radar-ui-engineer) / `etf-stocks.json` 섹터매핑(toss-data-engineer).

### 1-2. 단계 로드맵
| 단계 | 내용 | 전제 | 게이트 |
|---|---|---|---|
| **A** 기여도 분해 | 일봉으로 시장/섹터/고유 분해 → Y축·색 반영 | 현 데이터(추가 0) | G1 |
| **M** 멀티모달 fusion | 분봉·뉴스·수급 *측정용* 결합(탐지 lift) | 분봉 적재·뉴스·수급 확보 | 각 모달 잔존 설명력·재현성 |
| **P** 예측 | (봉인) 수익 예측 brain | 별도 라운드테이블 재합의 | 디폴트 NO |

### 1-3. 피저빌리티 게이트 G1 (A→M)
1. Look-ahead 차단(모든 baseline·β는 t 이전; 분해도 동일).
2. Walk-forward(rolling 재추정, in-sample 전체적합 금지).
3. 시장통제 후 잔존(고유 50%가 윈도우 무관 재현).
4. 거래비용·유동성(사후 진단 시 차감, 소형주 필터).
5. 재현성(결정론적 빌드, 버전 고정).

### 1-4. 결정 필요
- 섹터 정의: `themes[0]` 단일 vs 다중 가중.
- 시장 프록시: 유니버스 평균 vs KOSPI/KOSDAQ 지수.
- Y축 σ 정규화 정합(현 코드 raw/14 → σ 정규화 권고).
- 하위호환: 블립 append-only(누적/리플레이 모드 유지).
- 매니페스토: ledBy·shareSpecific은 *설명*일 뿐 — UI 카피에서 신호·예측 단어 금지.

**[아키텍처 결론]** A는 데이터 추가 없이 즉시 통합 가능. 핵심은 "Y축=고유 잔차, 블립 6→9 append, 색=ledBy" — 시장 따라간 종목은 중앙으로 사라지고 고유 이상치만 풀채도로 남아 SBV의 '편성≠연출'이 화면에 재현된다. G1 통과 전 fusion 진입 금지, 예측은 봉인.

## Part 2. Stock Behavior Vector — 신호/피처 (signal-quant)

### 2-1. 피처 8그룹 (SBV 6모달 대응)
| 그룹 | (SBV 대응) 피처 | 현 데이터 |
|---|---|---|
| **G1 가격액션** | log-return, 고저스프레드/ATR, 봉형태(body/shadow), 갭 | ✅ |
| **G2 거래량 미시구조** | RVOL, 거래량 z, VP상관, VWAP이탈 | RVOL·z ✅ / VWAP은 분봉 필요 |
| **G3 변동성** | 실현변동성(5·20일), 단기/장기 비율, ATR | ✅ |
| **G4 모멘텀·추세** | 1·5·20일 수익률, 52주 분위, 모멘텀 가속 | ✅ |
| **G5 섹터/시장 맥락** | 시장베타, 섹터평균, 종목-섹터 상관, **고유잔차(A)** | 섹터분류 ✅ / 지수 추가 |
| **G6 캘린더/이벤트** | 실적 전후 더미, 배당락, 지수편출입, 월·분기말 | 날짜파생 ✅ / 실적일 추가 |
| **G7 수급** | 외인·기관 순매수, 프로그램, 대차잔고 | ❌ KRX 등 추가 |
| **G8 뉴스/텍스트** | SBERT 감성, 특이키워드(공시·소송·계약), 빈도급증 | ❌ 뉴스+NLP 추가 |
- 투자 우선순위: **G5(섹터맥락 분해) > G2(거래량 z) > G7(수급) > G8(뉴스)**. (G8은 노이즈 비율 높아 후순위)

### 2-2. 기여도 분해 산식 (권고: 횡단면 OLS 잔차)
```
ret_{i,t} = α_t + β_t·ret_market,t + Σ_k γ_{k,t}·sector_dummy_{i,k} + ε_{i,t}
종목 고유 = ε_{i,t}  (이상점수에 이것만 사용)
```
- 베타조정 대비 장점: 비정상성·lag에 강건, 당일 공통움직임 당일 차감. A 실증("50% 시장·섹터")과 정합.
- 주의: 섹터 내 종목<5면 더미 불안정(상위섹터 합산). 장중엔 전일까지 계수 고정(look-ahead 차단).

### 2-3. 멀티모달 fusion 측정 프로토콜
- **타깃 정의 문제 명시**(주식엔 정답 라벨 없음). 대리 타깃 택1:
  - A: 이후 n일 실현변동성 급등 / B: 이후 n일 수익률 절댓값 상위 5%. (둘 다 사후 확인 가능 객관기준, 예측 아님)
- **lift 측정**: Baseline(G1+G3) → +G2 → +G7 → +G8 순으로 Precision@K·Recall@K·**ΔAUROC** 기록.
- **합격선**: 채널 추가당 ΔAUROC ≥ +0.03 채택, 미달이면 보류. 최소 6개월 out-of-sample, 학습/테스트 엄격 분리. 단독<조합이면 해당 채널은 노이즈로 제외.

### 2-4. 이상점수 재설계
```
anomaly = w1·|ε_z| + w2·rvol_z + w3·vol_ratio + w4·sector_divergence
  ε_z              : 횡단면 회귀 잔차 z (고유 이탈) — 주력
  rvol_z           : 거래량 z (자기 평소 대비)
  vol_ratio        : 단기/장기 변동성 비율
  sector_divergence: |ε_i − median(ε_sector)|
  초기 가중 w = 0.45 / 0.30 / 0.15 / 0.10 (상수 노출·튜닝 가능)
```
- 색칠/경보 임계: <1.5σ 정상(무색) · 1.5~2.5σ 주의(황) · 2.5~3.5σ 이상(주황·패널) · >3.5σ 극단(적·알림).
- 경보 근거 우선순위: ①고유 이탈 ②거래량 급증 ③섹터 내 역행 ④[실적일 D±n] 기저노이즈 경고.

### 2-5. fail-fast 게이트
- **G1 Look-ahead**: 당일종가로 당일신호 금지, 장중은 전일계수 고정. 날짜인덱스 겹침 assert.
- **G2 Survivorship**: 분석시점 상장종목만, 상폐 포함(불가시 '생존편향 있음' 명시·상방편향 추정).
- **G3 Walk-forward**: 최소 12개월·3개월 롤링, 윈도우별 독립 추정.
- **G4 거래비용**: lift 평가 시 스프레드·시장충격 차감. 소형주 스프레드 1~2%면 lift 소멸 가능 → 유동성 필터 선행.
- **G5 NULL 폐기**: walk-forward AUROC<0.55 전체폐기 / 채널 ΔAUROC<0.03 제외 / Precision@20<30% 실용불가 / 일일 이상비율>15% 역치 재보정.

**[신호 설계 결론]** 핵심 전환은 **절대 이탈 → 시장·섹터 차감 후 고유 이탈** 하나. 현 robust-z는 시장 전체 하락을 종목 이상으로 오탐하는 구조적 결함이 있고, A의 3분해를 횡단면 OLS 잔차로 구현하면 직접 해소된다. 현 데이터로 G1~G5 가능, G7·G8은 데이터 전제. fusion lift는 대리 타깃의 임의성을 *숨기지 말고 명시*(측정가능성 원칙). G1(look-ahead)·G2(생존편향)는 구현 전 데이터 레이어에서 먼저 닫아야 — 열려 있으면 이후 모든 측정이 무효.
