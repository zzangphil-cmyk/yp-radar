# 데이터 분석 — 441종목 × 21피처 (signal-quant + Karpathy)

> 데이터: `data/stock-features.json` (441종목, 오늘 단면 1개 + 종목별 200일 히스토리 산출). 스크립트: `prototype/analyze.mjs`.
> 성격: **서술적 분석(오늘 누가 이상한가)**. 예측·알파 아님(단면 1개라 타깃 없음).

## 1. 죽은 컬럼 + 중복 — "21피처"는 사실 ~8차원
- **죽은 컬럼(분산 0, 즉시 제거)**: `lev`(전 종목 0), `tradeN`(전 종목 50, 수집 상한). → 마할라노비스 전에 *반드시* 제거(안 하면 Σ 특이·NaN).
- **정의적 중복(|ρ|>0.7)**: mktCap≈turnover(0.98) · body≈ret1(0.83) · vol20≈atr14(0.82) · ret5≈ret20≈ret60≈pos200(0.55~0.77 사슬).
- **유효 정보 차원**: participation ratio = **8.55** (Kaiser λ>1 → 6개). 90% 분산에 PC 9~10개. → "21차원"은 포장, 진짜는 ~8축.

## 2. relVol은 진짜 독립 신호
- relVol ⟂ turnover(ρ=0.00), ⟂ vol20(ρ=0.03) — 절대 거래량과 구조적으로 다름.
- relVol 테마 R²=**0.003** → 섹터 통제 거의 불필요한 *유일* 피처. 반대로 vol20 테마 R²=0.155(시총 편향) → 반드시 통제 후 잔차.

## 3. 시장·섹터 통제 후 고유
- ret1 테마 R²=**0.068** → 당일 수익률의 93%가 종목 고유. **고유 잔차가 핵심 이상탐지 피처.**
- 오늘 고유 수익 상위: 피에스케이(+17.7%), 가온전선(+14.1%), 금호타이어(+13.4%).

## 4. ★ 조합(마할라노비스 D²)이 단일 피처가 못 잡는 걸 잡는다
핵심 6피처 {relVol, |고유ret1|, volRatio, obImbalance, |tradeStrength|, range} 표준화 → Ledoit-Wolf 공분산 → $D^2=z^\top\Sigma^{-1}z$. p/N≈0.04, 조건수 368 → **마할라노비스 정당**.

**오늘 D² 상위**: 금호타이어(406, relVol z=20.2) · 엠앤씨솔루션(82, range) · 리가켐바이오(69) · 피에스케이(47) · 가온전선(31) …

**Joint anomaly 실증 — OCI홀딩스**: 어떤 피처도 2σ 미만(z_max=1.84)인데 D²=12.1. obImbalance·volRatio·tradeStrength가 *동시에* 1.5σ대 → **단일 z 스크리닝은 놓치고 D²만 잡음.** 이게 "분해 후 조합"의 가치 — 당신이 지적한 빠진 조각이 실제 데이터에서 작동.

## 5. 최종 신호 처방
- 이상점수 = Ledoit-Wolf 공분산 **마할라노비스 D²**.
- 입력 6피처: **relVol · |ret1 고유잔차(테마 통제)| · volRatio · obImbalance · |tradeStrength| · range**.
- 제거: lev·tradeN(죽음), turnover·mktCap·body·atr14·ret5/20/60(중복·시총편향).
- 효과: "우량주·잡주 어디서나 3σ가 같은 의미" 공정성 보장.

## 6. 정직한 경계 (Karpathy brake — 치명)
- ✅ **할 수 있는 것**: 오늘 단면에서 "누가 통계적으로 이상한가" *서술*. 마할라노비스·robust z 정당.
- ❌ **할 수 없는 것**: 예측·알파·백테스트·임계치 튜닝. **타깃 컬럼이 없다.** 단면 1개라 train/test 분리 불가 → 어떤 컷도 오늘에 과적합.
- ⚠️ **착시 1순위**: 과거 ret(이미 피처)를 라벨로 끌어다 "예측" 주장 = 누설(leakage). 서술과 예측을 섞는 순간 함정.
- → 예측 검증은 *다른 날 데이터를 누적*해 시간 분리한 뒤에만(#3). 지금은 "오늘 누가 이상한가"까지만.

## 결론
**"분해 + 조합"이 실제 데이터에서 작동함을 확인** — 6개 정제 피처의 마할라노비스 D²가 단일 신호가 놓치는 복합 이상치(OCI)를 잡는다. 단 이건 *서술적 이상탐지*이지 예측이 아니며, 21피처는 실제 ~8차원이다. 다음 가치는 **시계열 누적(#3)** — 그래야 예측 검증이 가능해진다.
