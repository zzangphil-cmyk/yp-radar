# 미세 신호 심층 — "버린 작은 신호 다시 보기" (SBV 원칙1)

> signal-quant(정량) + CSO(SBV 철학). 데이터 `data/stock-features.json`, 스크립트 `prototype/analyze2.mjs`.
> 동기: 사용자 — "SBV에서도 미세 신호가 다른 맥락에선 의미였다(원칙1: 작은 신호 무시 금지)."

## 1. 버린 중복쌍의 직교 잔차 — 일부는 진짜 구조 신호
중복(상관 높음)이라 버린 부분의 *잔차(orthogonal residual)*를 검사. 대부분 단면 이상점수와 약하나(ρ_D²<0.09), **두 개는 살려야 함**:

- ★ **turnover ⊥ mktCap = "순수 자금유입"** — 전체 ρ는 낮지만 극단이 극명: **SK하이닉스 z=+13.65**(시총 대비 거래 폭증=자금 쏠림), **삼성전자 z=−12.81**(시총 1위인데 거래 부족). "대형주 중 어디에 실제 자금이 쏠리나"를 단면에서 분리. **구조적 real.**
- ★ **ret20 ⊥ ret5 = "모멘텀 지속성"** — ρ_D²=0.089(최고). 피처셋에 추가하면 신세계·SK하이닉스·삼성전기가 대폭 상위로(단기 평범·중기 강세 패턴). **구조적 real.**
- body⊥ret1·atr14⊥vol20 — 약함, 극단 케이스만(reasons 패널 보조 텍스트용).

## 2. 조건부(strata) — 가설 단계
소형/대형·코스피/코스닥별 약신호(호가불균형·스프레드·체결강도)의 이상관련성: 모두 ρ<0.2 미달. 단 **소형주×고RVOL에서 obImbalance ρ가 0.02→0.09(5배)** — 방향은 맞으나 절대치 낮음. → **가설(시계열 n≥10 누적 후 확인).** 단면 자동 적용 금지(오탐).

## 3. 미세 다중 vs 단일 극단 — joint anomaly 7종목 실존
z_max ↔ D² ρ=0.905(높음) → D² 상위 대부분은 단일 피처도 이미 돌출. 그러나 **순수 joint anomaly(z_max<2 & D²≥P90) = 7종목(1.6%)**: OCI홀딩스·SKC·제일기획·LG유플러스·에임드바이오·미스토홀딩스·티에스이 — 각 3~4피처가 1.5σ 전후 동시. **단일 z 스크리닝이 못 잡고 D²만 잡음 = 조합의 가치.** (단정 라벨 금지, "복수 신호 동시 이탈"로 표기)

## 4. ★ CSO 철학 — 살릴 신호의 판별 칼
SBV에서 **진짜 작은 신호(fusion +0.1157·engagement 14.35x·b=0.498)**와 **사라진 직관(몽타주·midpoint·H-D1 6기각)**을 가른 칼:

> **살아남은 건 "맥락 통제 후 잔차"에서 컸다. 죽은 건 "맥락이 만든 단면 착시"였다.**

주식 적용:
- **살린다**: 시장·섹터·자기과거(within) 통제 후 *잔차*에서 부호·크기가 남고, 여러 단면(시간·종목)에서 √N처럼 반복되는 것.
- **기각한다**: 단면 하나의 그럴듯함·사후 서사(올랐으니 강했겠지)로만 큰 것 — 아무리 작아도.
- **미시구조(호가·체결·스프레드)**: Audio(5D, 작지만 의미)일 수도, '연출'(착시)일 수도 — **미리 단정 말고 시험을 치르게**(통제 후 잔차 생존 여부). raw 호가불균형은 시장 베타이므로 *잔차 불균형*만 신호.

**Karpathy(단면 함정) vs 원칙1(작은 신호) 양립**: 같은 적(미검증 신호)을 가리킴.
> **버리는 문턱은 낮게(쉽게 선험 제거 말 것) · 믿는 문턱은 높게(여러 단면 통과해야 믿을 것).** 작은신호 살리기=*진입* 규칙 / 단면경계=*승급* 규칙. 층위가 다르니 충돌 없음.

⚠️ **경고(CSO)**: 6피처 축소 자체가 "좋은 차원만 합치면 낫다"는 *산술 덧셈(원칙2)* 냄새. 상관 높다고 무가치 아님 — fusion gain은 상관 높아도 결합 증분이 났다. 줄이기 전 "교호항이 잔차를 설명하나"를 물었어야.

## 5. 교호항 정식 검정 — CSO 경고 입증 (analyze3.mjs / result-interaction.md)
"6피처 축소가 산술덧셈(원칙2)인가"를 정식 검정: 중복쌍 (A,B)의 *직교화 교호항* A·B가 6피처 D²를 추가 설명하나(ΔR²), 순열검정 nPerm=2000·부트스트랩 nBoot=500.

| 중복쌍 | ΔR² | 순열 p | 판정 |
|---|---|---|---|
| ★ **body × ret1** | **0.081** | **0.001** (z=11.67) | **복원** — 봉몸통·수익률 *동조 강도* 자체가 독립 이상정보. 일방향 결집봉 vs 갭·수염봉 구분 |
| ★ **ret5 × ret20** | 0.008 | 0.034 | **복원** — 모멘텀 *가속*(단·중기 동시 극단). 07의 ret20⊥ret5 잔차와 일관 |
| mktCap × turnover | — | 0.072(경계) | 교호항 불필요, 단 *선형 잔차*(자금유입)는 07대로 유지 |
| vol20 × atr14 | 0.0004 | 0.676 | 버려도 정당 (D²와 ρ≈0) |
| ret5×ret60·ret20×ret60 | ≈0 | >0.76 | 버려도 정당 |

- body×ret1 복원 시 6피처가 놓친 종목 신규 포착: **스피어(방산 257→9위)·기가비스(45→10)·삼성물산우B(297→20)**.
- 전체(C: 교호항+07잔차) 모델은 삼성전자(285→3)·SK하이닉스(385→2)를 상위로 — 자금유입 잔차가 대형주 쏠림 포착.
- ⚠️ **소표본 경고**: body×ret1 부트스트랩 SE=0.080 ≈ ΔR² 수준. 단면 1개의 *서술* 개선이지 예측 아님. 날짜 누적 후 재검정 필수.

→ **CSO가 옳았다**: 6피처 축소는 *부분적으로 부당*. body×ret1·ret5×ret20 두 교호항은 fusion-gain처럼 결합 증분이 실재 → 복원. vol20×atr14·장기 ret쌍은 진짜 중복 → 폐기 정당.

## 결론 — 모델 업데이트
1. **잔차 신호 2개 복원**: `자금유입(turnover⊥mktCap)` + `모멘텀지속(ret20⊥ret5)` — 통제 후 살아남음 → 피처/이상점수에 추가(극단 클리핑 동반).
2. **교호항 2개 복원**(§5): `body×ret1`(동조강도) + `ret5×ret20`(모멘텀가속) — 직교화 후 D² 입력. vol20×atr14·장기 ret쌍은 폐기 정당.
3. **D² 유지** — joint anomaly 7종목은 D²만 포착.
4. **조건부 미세신호(호가·체결 strata)** → **승급 트랙**: 시계열 누적(#3) 후 within·시간 통제 생존 시 채택. 단면 자동적용 금지.
5. **원칙**: 진입 쉽게 / 승급 어렵게. 차원 축소를 임의 임계값으로 하지 말 것 — 버리기 전 *교호항 검정*을 거칠 것(§5가 그 절차).
6. ⚠️ **단면 1개 한계**: 모든 ΔR²·p는 *서술* 개선이지 예측 아님. body×ret1 SE≈ΔR² → 날짜 누적 후 재검정 필수.
