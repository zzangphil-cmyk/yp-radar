## 한 줄 결론
**예측은 불가능(증명 3회), 가치는 관측에 있다.** 가격·거래량·표준지표에서 거래가능한 예측 알파는 없다(EMH·McLean-Pontiff 정합). 그래서 제품은 *예측기*가 아니라 **"지금 무슨 일이, 왜, 통념은 뭐라 하는지"를 정직하게 조립해 사람이 판단하게 하는 곡괭이**다.

## 배포된 제품
**온도계 레이더** + **판단 근거 카드** (`src/components/StockRadar.tsx`, `scripts/build-radar.mjs`)
- universe = 코스피 200 + 코스닥 50 (250종목), 시장 내 표준화
- **온도 = 마할라노비스 D²**(5축: 거래량·고유수익·변동성·당일폭·자금유입, Ledoit-Wolf), 로그 압축 눈금. "평소와 얼마나 다른가"이지 방향 아님.
- **판단 근거 카드**(종목 선택 시): ①왜 떴나(시장/섹터/고유) ②무엇이 특이(5축 시장내 백분위) ③통념 지표(RSI·MACD·볼린저…)+**"검증상 예측력 0" 정직 라벨**
- 데이터 파이프라인: `build-panel`(검증용 패널) · `build-radar`(온도·백분위) · `build-ta-panel`/`build-ta-latest`(통념지표) · `collect-disclosures`(이벤트 검증)

## 정직한 경계 (변하지 않음)
- **이상 ≠ 매매신호 · 투자자문 아님.** 예측·알파·백테스트 수익 주장 금지.
- 단면·소표본(140일·intraday) 한계 명시. 동시·후행 이벤트 연관은 *거래 edge 아님*.
- 아직 안 본 곳: 미시구조 누적(선행 신호 재도전, 오늘부터)·펀더멘털. 가격 파생은 세 번 다 null.
- 자문 페르소나(Musk/Thiel/Karpathy/Ott-CTO)는 공개자료 기반 추정이며 실제 인물 의견 아님.
