# SBV → 주식 레이더 적용 — 피저빌리티 검토

> **상태: 피저빌리티(타당성) 검토.** 프로덕션 배포 아님. 결론·게이트 통과 전 실서비스 반영 금지.
> 작성: 2026-06-28 · 대상: Y&P 레이더 `/radar` · 출처 방법론: SBV(View Pulse, `D:\SBV_model\viewpulse-analysis`)

## 무엇인가
SBV(Scene Behavior Vector — 한국 방송 콘텐츠 멀티모달 시청률 brain) 방법론을 **주식 종목 레이더**에 적용하면 어떤 결과가 있을지 검토한 기록.

## 한 줄 결론
**"예측 brain은 No, 방법론 이식은 Yes."** SBV의 시청률 R²0.98은 주식에 transfer되지 않는다(효율시장·노이즈·재귀성). 그러나 SBV의 진짜 자산 — **상대화 · 기여도 분해 · 멀티모달 fusion 측정 · fail-fast** — 는 정직하게 이식돼 레이더를 **"왜 떴나(시장·섹터 vs 종목 고유)"를 분해하는 정직한 이상 스캐너**로 강화한다. 실증: 현 이상치의 **약 50%가 시장·섹터 동반**(가짜에 가까움), 50%만 종목 고유.
