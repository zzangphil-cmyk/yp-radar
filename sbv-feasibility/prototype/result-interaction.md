> 스크립트: `sbv-feasibility/prototype/analyze3.mjs`

## 배경 및 검정 질문

06 분석에서 21피처를 6피처로 줄이며 네 개의 "중복쌍"을 버렸다.
CSO 경고: "상관 높다고 무가치 아님 — SBV fusion gain은 상관 높은 모달리티끼리도 결합 증분이 났다."

**검정 질문**: 각 중복쌍 (A, B)의 교호항 A·B가 D²(서술적 이상강도)를 추가로 설명하나?
ΔR²이 순열 유의하면 "그 쌍은 버리면 안 되는 결합 정보". 무의미하면 "진짜 산술적 중복 → 축소 정당".

---

## 검정 방법

1. **타깃**: 6피처 Ledoit-Wolf 마할라노비스 D²(표준화). 단면에 라벨 없으므로 D²를 서술적 이상강도 대리변수로 사용.
2. **모형 비교**: 가산 모형 `Y ~ 1 + A + B` vs 교호항 포함 `Y ~ 1 + A + B + A·B_orth`
   - A·B는 직교화(A,B 선형분 제거)해 순수 비선형 상호작용만 추출 → 다중공선성 억제
3. **유의성 검정**: 순열검정 nPerm=2000 (Y를 섞어 ΔR² 영분포 구성)
4. **강건성**: 부트스트랩 nBoot=500 (복원추출), 95% CI
5. **후속 분석**: 유의 교호항 포함 D² vs 기준 D² 순위 비교 (TOP20 기준 종목명 비교)

## 2. 쌍별 상세 해석

### (A) body × ret1 — 교호항 복원 필요 ★★★

- **ΔR² = 0.08073, 순열 p = 0.001, z-stat = 11.67**
- 6피처 기준 D²의 8.1%를 추가로 설명. 순열 z-stat 11.67은 영분포(SE=0.0068)와 극명하게 다름.
- **부트스트랩 95% CI: [0.0558, 0.3106]** — 0 미포함, 강건.
- 교호항 β₃ = +0.195: body와 ret1이 **같은 방향으로 동시에 클 때**(고수익 + 빡빡한 양봉, 또는 급락 + 음봉 꽉 참) 이상강도가 추가로 높아짐.
- 06에서 "봉 몸통과 수익률은 같은 움직임"이라 버렸으나, 교호항이 보여주는 것은 **그 두 신호의 동조 강도(빡빡함)**이 이상탐지에 독립적 정보임.
- 직관: 수익률은 같은데 봉이 꽉 찬 종목(체결 압박)과 봉이 헐거운 종목(갭·수염)은 이상성 성격이 다름.
- **단, 잔차↔D² ρ=0.028로 낮음** — 교호항 자체(비선형 결합)가 중요하지, 잔차 선형 신호로는 약함.

### (B) ret5 × ret20 — 교호항 복원 필요 ★

- **ΔR² = 0.00802, 순열 p = 0.034**
- 부트스트랩 95% CI: [0.0014, 0.1827] — 0 미포함, 강건.
- 잔차↔D² ρ = 0.089 (07에서 이미 "모멘텀 지속성"으로 살린 신호와 일관).
- 교호항 β₃ = +0.055: 단기·중기 수익률이 같은 방향으로 **동시에 클 때** 이상성 증가.
- 해석: 단기(5일)와 중기(20일) 수익률이 모두 극단적이면 → 단순 추세 이상이 아닌 **모멘텀 가속** 신호. 단기만 튀거나 중기만 좋은 것은 이상이 아니나, 둘이 동조해 극단이면 이상.
- 07에서 잔차(ret20⊥ret5)를 모멘텀 지속성으로 살렸는데, 교호항 검정이 이를 **추가로 뒷받침**. 일관적.

### (C) mktCap × turnover — 잔차 유지(경계, p=0.072)

- ΔR² = 0.00209, p = 0.072. 유의 기준(0.05) 미달, 경계.
- 잔차↔D² ρ = 0.029 — 낮음.
- **그러나 07에서 이미 잔차 신호(순수 자금유입)를 구조적으로 살림**: SK하이닉스 z=+13.65, 삼성전자 z=-12.81. 이는 **교호항이 아니라 잔차 선형 신호**로 작동하는 것.
- 결론: 교호항 복원은 정당화되지 않으나, **잔차(turnover⊥mktCap) 자체는 07의 판정대로 유지**.

### (D) vol20 × atr14 — 버려도 정당

- ΔR² = 0.00035, p = 0.676. 유의성 없음.
- 잔차↔D² ρ = -0.000. 이상탐지 기여 없음.
- 07에서 atr14⊥vol20 잔차를 "가격이동 효율성"으로 살렸으나, **이 단면에서 D²와의 연관이 사실상 0**.
- 결론: 교호항 복원 정당화 안 됨. 잔차 신호도 약함(보조 텍스트 수준).

### (E) ret5 × ret60, ret20 × ret60 — 버려도 정당

- ΔR² 모두 0.0001 미만, p > 0.76. 명백히 무의미.
- 추세 사슬(단기↔장기)의 중복은 교호항도 정보 없음.

## 3. 피처셋별 D² 이상점수 순위 비교

**세 피처셋:**
- **A**: 기존 6피처 (기준)
- **B**: 기존 6피처 + 유의 교호항 2개 (body·ret1_orth, ret5·ret20_orth)
- **C**: B + 07 잔차 2개 (turnover⊥mktCap, ret20⊥ret5)

| 비교 | 순위 상관(스피어만) | 해석 |
|---|---|---|
| A ↔ B | ρ = 0.855 | 교호항 추가로 15% 순위 변동 |
| A ↔ C | ρ = 0.777 | 풍부한 모델에서 23% 순위 변동 |

### TOP20 진입/이탈 종목 (A vs B 비교)

**A와 B 공통 16종목**: 제닉, 리가켐바이오, 원익IPS, 코스메카코리아, 케이씨텍, 하나기술, 피에스케이, 두산퓨얼셀, 엠앤씨솔루션, 유진테크, 가온전선, 테스, 피에스케이홀딩스, 브이엠, 금호타이어, SK디스커버리

**A에만 있었으나 B에서 밀려난 4종목 (교호항 추가로 상대적 이상성 약화)**:
→ 한국콜마, 카카오게임즈, **OCI홀딩스**, 한국쉘석유

**B에서 새로 포착된 4종목 (교호항이 잡아낸 이상)**:
→ **스피어, 기가비스, 신성델타테크, 삼성물산우B**

### A vs C (가장 풍부한 모델)로 새로 포착된 9종목:
→ **스피어, 에이프릴바이오, SK스퀘어, 기가비스, SK하이닉스, 삼성전자, 삼성전기, 삼성생명, 현대백화점**

주목: C 모델에서 삼성전자(A위=285→C위=3), SK하이닉스(A위=385→C위=2)가 상위 진입.
→ 이는 07에서 확인한 "turnover⊥mktCap 순수 자금유입" 잔차 신호가 대형주 내 자금 쏠림을 잡아냈기 때문.

### 교호항이 새로 포착한 종목 상세 프로파일

**스피어 (방산)**: 6피처에서 A위=257이던 종목이 B에서 9위로. D²_A=3.50 → D²_B=26.71.
기본 z는 전부 낮음 (relVol z=-0.20, absRet1 z=-0.70). 교호항 효과로만 이상성 표면화.

**기가비스 (코스닥)**: A위=45→B위=10. D²_A=8.08 → D²_B=25.92.
absRet1 z=+1.69, absTrStr z=+1.77. 두 피처가 중간 수준으로 동조하는 패턴.

**삼성물산우B (배당·커버드콜)**: A위=297→B위=20. 기본 z 전부 낮음, 교호항만이 이상을 포착.

## 4. 소표본 과적합 경고 및 한계

- **N=440, 단면 1개**. 순열 p는 이 단면에서의 구조적 일관성이지 시계열 예측력이 아님.
- **body×ret1의 ΔR²=0.081은 크지만**, 부트스트랩 SE=0.080으로 ΔR²과 비슷한 수준. CI가 [0.056, 0.311]로 매우 넓음 → 실제 효과 크기의 불확실성 높음.
- **ret5×ret20의 ΔR²=0.008은 작음**. CI [0.001, 0.183]이 0은 포함하지 않으나 상단이 매우 불안정.
- 모든 교호항 결론은 "이 단면에서 이상 서술이 개선된다"이지 "미래 이상을 더 잘 탐지한다"가 아님.
- 타깃(D²) 자체가 6피처로 구성되어 있어 tautology 위험: body, ret1이 D² 계산에 직접 영향을 미치지 않으므로(6피처에 미포함) 이 검정은 valid하나, 향후 타깃 독립성 재검토 권장.

## 최종 한 줄 판정

**6피처 축소는 부분적으로 정당하지 않다: body×ret1 교호항(ΔR²=0.081, p=0.001)과 ret5×ret20 교호항(ΔR²=0.008, p=0.034)은 기존 6피처 D²가 잡지 못한 이상종목(스피어·기가비스·삼성물산우B 등)을 포착하므로 이 두 교호항을 직교화하여 복원해야 하며, vol20×atr14 및 ret계열 장기쌍은 교호항도 잔차도 기여가 없어 버려도 정당하다. 단, 이상의 결론은 단면 1개의 서술적 이상탐지 개선이지 예측력이 아니며, 부트스트랩 SE가 크므로 추가 날짜 누적 후 재검정이 필수다.**
