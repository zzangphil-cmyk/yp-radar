# 모델 검증 결과 (정직한 검증)

> 데이터: 최근 30거래일 (05/14~06/26), 50종목 일봉. **소표본 — 예측력 결론용 아님, 방법 타당성 검증.**

## V1. 분해 검증 — 시장·섹터가 종목 변동을 얼마나 설명하나 (서술적, 유의미)
- 시장만 평균 R²: **46.846%**
- 시장+섹터 평균 설명비: **46.579%**
- 평균 **고유(종목 특이) 비중: 53.421%**

→ 종목 변동의 약 46.579%가 시장·섹터로 설명됨. **분해 전제(시장·섹터를 빼야 고유가 보인다) 확인.** SBV의 'Program+Context 지배'와 같은 구조.

## V2. 반증 검증 — 고유 이상이 미래와 관계있나 (룩어헤드 차단)
- 고유수익 1차 자기상관 ρ(ε_t, ε_{t+1}) = **0.046**  (표본쌍 1450, SE≈0.026)
- 해석: ρ<0=평균회귀 / ρ>0=모멘텀 / |ρ|≲2·SE(0.053)=무의미(효율시장)
- 큰 고유 이탈(상위10%) 다음날 되돌림 비율: **49.655%** (50%=무작위)

→ 판정: **효율적(예측 신호 없음) — 효율시장과 정합.** 고유 이상은 *탐지*엔 유효하나 *예측*엔 신호 없음 — 라운드테이블 결론(예측 불가) 재확인.

## 결론
- **서술(분해)은 검증됨**: 시장·섹터가 변동의 46.579%를 설명 → "왜 떴나" 분해가 실제로 의미.
- **예측은 신호 없음(정직)**: 우리 모델은 *예측기가 아니라 관측·분해 렌즈*임이 데이터로 재확인.
- **한계**: 30거래일×50종목 소표본, 일봉, 거래비용 미차감. 본 검증은 *방법 타당성*이지 *시장 알파 주장*이 아님.
